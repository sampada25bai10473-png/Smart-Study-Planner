# Rule-Based Study Recommendation System

## Description
This project helps students create a personalized study plan based on subject difficulty and the number of days left before exams.

## Problem Statement
Students often face confusion when deciding how much time to allocate to different subjects during exam preparation. This project aims to solve that problem using a simple rule-based approach.

## Solution
The system takes input such as subject names, difficulty levels, and days left for exams. Based on predefined rules, it calculates and recommends the number of study hours for each subject.

## Features
- Accepts multiple subjects as input
- Considers difficulty level (weak/medium/strong)
- Adjusts study hours based on time remaining
- Calculates total study hours per day
- Provides helpful study tips

## Technologies Used
- Python
- Rule-Based Logic

## Setup Instructions
No additional setup is required. Just make sure Python is installed on your system.

## Requirements
- Python 3.x

## How to Run (Execution Steps)
1. Open terminal
2. Navigate to the project folder
3. Run the following command:
   python app.py

## Example
Input:
Math (weak), Science (medium), Days left: 3

Output:
Math: Study 5 hours per day  
Science: Study 3 hours per day
